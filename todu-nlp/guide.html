智办Guide

功能使用介绍

用自然语言创建待办，智能解析日期、时间、干系人。离线运行，数据本地存储，无需服务器。

自然语言创建待办

在输入框中用日常语言描述待办事项，系统会自动解析出事项名称、描述、干系人、截止时间、状态、备注。

支持的日期格式

  • 0617 — 月+日（当年）
  • 20260617 — 完整日期
  • 2026-6-17 / 2026/6/17 — ISO 格式
  • 6月20日 — 中文日期
  • 明天 / 后天 / 一周后 — 相对时间
  • 下周三 — 星期
  • 3天后 — 天数偏移

支持的时间格式

  • 9点 / 14:30 — 直接时间
  • 上午9点 / 下午3点 / 晚上8点 — 带时段

支持的属性关键词

  • 干系人：负责人是小王、由小王负责、分配给小王、@小王
  • 备注：备注需要财务数据、注意XXX
  • 状态：正在做 → 进行中、已完成 → 已完成、还没开始 → 未开始
示例 1：
0617下午3点前完成季度报告，负责人是小王，备注需要财务数据
→ 事项「完成季度报告」/ 截止 6月17日 15:00 / 干系人 小王 / 备注 需要财务数据
示例 2：
明天上午10点和客户开会，由张三跟进，注意带PPT
→ 事项「和客户开会」/ 截止 明天 10:00 / 干系人 张三 / 备注 带PPT
示例 3（@提及干系人）：
明天10点开产品评审会 @张三，注意需要PRD文档
→ 事项「开产品评审会」/ 截止 明天 10:00 / 干系人 张三 / 备注 需要PRD文档
示例 4（@多人）：
0618下午2点前提交代码审查 @李四 @王五
→ 事项「提交代码审查」/ 截止 6月18日 14:00 / 干系人 李四, 王五

解析结果会显示在下方可编辑预览区，你可以修改任意字段后再点击创建。

若未输入或未识别到日期，新任务的截止时间默认为今天 23:59，可在预览区修改。

粘贴图片创建待办

直接把图片粘贴到输入框（Ctrl/Cmd + V），图片会作为该待办的附件保存（不做文字识别），预览区会显示缩略图，可点击 ×移除。

  • 若只粘贴图片、未输入文字，任务名称默认为 图片任务 加当日日期，如 图片任务2026-06-18。
  • 若同时输入了文字，标题照常从文字解析，图片仅作为附件。
  • 列表中图片以缩略图展示，点击缩略图可浮窗查看大图（点图片或按 ESC 关闭）。
  • 图片会经压缩后存入本地，导出备份时随 JSON 一并保存，导入后可恢复。

筛选条件

通过筛选栏快速查看不同状态的待办，每个筛选前有彩色圆点区分：

未完成 今日到期 全部 未开始 进行中 已完成 已超期
  • 未完成（珊瑚色点）— 所有未完成的待办
  • 今日到期（珊瑚色点）— 今日需完成的待办（截止时间在今天内）
  • 全部（墨色点）— 所有待办
  • 未开始（金色点）— 状态为"未开始"的待办
  • 进行中（琥珀色点）— 状态为"进行中"的待办
  • 已完成（绿色点）— 已完成的待办
  • 已超期（红色点）— 已过截止时间但未完成的待办

任务卡片颜色

卡片左侧色条代表不同状态，背景色代表紧急程度：

青色色条 — 未开始状态
琥珀色条 — 进行中状态
绿色色条 — 已完成状态（标题有删除线）
琥珀色条 + 浅橙背景 — 今日到期（今天需完成）
红色色条 + 浅红背景 — 已超期

任务操作

每个任务卡片底部有以下操作按钮：

  • 设为进行中 — 将未开始的任务标记为进行中
  • 退回未开始 — 进行中的任务可退回未开始状态
  • 完成 — 标记任务为已完成
  • 重启 — 将已完成的任务重新设为未开始
  • 延期一天 — 已超期或今日到期的任务可顺延一天（仅在此时显示）
  • 设为循环 — 将任务设为每周 / 每月 / 每季度循环任务（任务默认非循环）
  • 完成并下一周期 — 循环任务专属，完成本期并把截止时间推进到下一周期、状态重置为未开始
  • 取消循环 — 将循环任务恢复为普通任务
  • 编辑 — 弹窗编辑所有字段（名称、描述、干系人、截止时间、状态、备注）
  • 删除 — 弹出自定义确认弹窗，确认后才会删除（取消则数据不变）

循环任务

对于按周、按月、按季度重复的任务（如周报、月度复盘、季度总结），可点击卡片上的 设为循环 按钮，选择周期后任务会带上 ↻ 每周/每月/每季度循环 标签。

  • 循环任务的"完成"按钮会变为 完成并下一周期，点击后截止时间从原截止日自动推进一个周期、状态重置为未开始，实现"完成本期即进入下一期"。
  • 无需新建任务，同一条任务直接跳转到下一周期，列表保持干净。
  • 如需彻底结束循环，先点 取消循环 恢复为普通任务，再点完成即可。

快捷键：ESC 关闭弹窗，输入框中 Enter 创建待办，Shift+Enter 换行。

搜索

筛选栏右侧有搜索框，输入关键词即可在当前筛选结果中进一步搜索，支持匹配事项名称、描述、干系人、备注。

数据存储与备份

存储方式

数据存储在浏览器的 localStorage 中，直接双击 index.html 即可离线使用，无需安装任何服务器。

导出备份

点击右上角 导出备份 按钮，将所有待办数据导出为 JSON 文件，保存到本地。

系统会每 7 天自动提醒你导出备份，防止浏览器数据清理导致丢失。

恢复导入

点击右上角 恢复导入 按钮，选择之前导出的 JSON 文件。支持三种模式：

  • 本设备合并 — 同一设备的备份恢复，按 id 去重后追加到当前列表。
  • 跨设备合并 — 合并来自其他设备的备份，按内容（标题+截止时间+干系人）去重，适合多设备同步。
  • 覆盖导入 — 清空当前数据，用文件内容替换。
建议：定期导出备份（至少每周一次），将 JSON 文件保存到安全位置。如遇浏览器数据清理，可通过导入功能完整恢复。

主题切换

点击右上角的月亮 / 太阳图标可在浅色与深色主题间切换，偏好会保存在本地，下次打开自动恢复，并在主页与介绍页之间保持一致。

  • 首次使用时，若未手动切换，将跟随系统的深浅色偏好（prefers-color-scheme）。
  • 切换时浏览器地址栏 / 状态栏的配色也会同步变化。
  • 主题切换仅改变外观，不影响任何待办数据。

快速上手流程

  • 在输入框用自然语言描述待办（如：明天下午3点开会，负责人李四）
  • 检查下方解析预览，按需修改字段
  • 点击「创建」按钮添加待办
  • 通过筛选栏查看不同状态的待办
  • 点击任务卡片的操作按钮管理状态
  • 定期点击「导出备份」按钮保存数据
